sebelum menggunakan, sesuaikan terlebih dahulu php.ini di xampp. copy file php.ini github saya ke php.ini kalian

setelah php artisan make:filament-user, lakukan php artisan tinker dan jalankan perintah di bawah ini satu persatu :

use App\Models\User;
use Spatie\Permission\Models\Role;
$user = User::where('email', 'email_user@example.com')->first();
$role = Role::firstOrCreate(['name' => 'super_admin', 'guard_name' => 'web']);
$user->assignRole($role);
echo "User berhasil diberikan role super_admin!";
